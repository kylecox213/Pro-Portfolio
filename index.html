This application allows users to find a friend based upon compatability. The degree of compatability is determined by the total score derived a questionaire that each user must complete. This application enables users to create a burger. Users will enter the name of their custom burger and click the 'create' button. The burger name will be rendered in the 'Concept Burgers' table accompanied by an 'order up' button. Clicking the 'order up' button will remove the burger from the 'Concept Burgers' table and render to the 'Signature Burgers' table.
